# 📢 ComplaintGram
### Instagram for Civic Complaints — AI-Powered, Authority-Routed, Publicly Accountable

> Built for India. Powered by Gemini AI. 
> Because complaints shouldn't disappear into a black hole.

## 🌐 Run Locally
```bash
pip install streamlit google-generativeai
streamlit run app.py
```

## 🚨 The Problem
Every day millions of Indians face broken roads, water shortages, 
garbage overflow and power cuts. They complain on GHMC apps, 
Twitter, WhatsApp groups — and nothing happens.

Why? Because complaints are:
- Informal and unrouted
- Invisible to the public
- Easy for authorities to ignore
- Never converted into formal actionable documents

## 💡 The Solution
ComplaintGram combines social media visibility with AI intelligence:

**You post → AI analyzes → Authority gets routed → Public sees → 
Pressure builds → Problem gets fixed**

## ✨ Features

### 📰 Social Feed
- Instagram-style complaint feed
- Real complaints with proof attached
- Upvote to show community support
- Filter by city, category, severity
- Live stats — total complaints, critical issues, upvotes

### 🤖 AI Analysis (Gemini powered)
- Severity scoring 1-10 with reasoning
- Automatic authority routing (GHMC, HMWSSB, Police etc)
- Escalation path if ignored
- Legal rights of the citizen
- Estimated resolution timeline

### 📄 Formal Letter Generation
- Complete professional complaint letter generated instantly
- Ready to print and submit
- Addressed to correct authority with all details

### 📱 Social Pressure Tools
- Tweet generator with hashtags
- WhatsApp message for colony groups
- Hashtag suggestions for maximum visibility

## 🏛️ Smart Authority Routing
| Problem | Authority |

| Water Supply | HMWSSB |
| Electricity | TSSPDCL |
| Traffic Signals | Traffic Police |
| Pollution | Telangana PCB |
| Illegal Construction | GHMC Town Planning |

## 📊 Why This Matters
- 500M+ Indians face civic issues daily
- Existing apps (GHMC, MyGov) have no public visibility
- Complaints are informal — easy to ignore
- No tool converts complaints into formal legal documents
- No platform aggregates community pressure

ComplaintGram fills all these gaps in one place.
